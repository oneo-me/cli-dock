# Changelog

## 1.3.0

- Added a Run grok button that launches the `grok` command in the current editor group.

## 1.0.1

- Added a GitHub repository button that opens the current project's GitHub remote.
- Switched project package management from pnpm to bun.
- Updated lazygit to use separate light and dark icons.

## 1.0.0

- Initial release with editor title buttons for lazygit, opencode, codex, and claude.
- Launches each command in an integrated terminal in the current editor group.
